class html2ans.parsers.base.ParseResult[source]¶

Bases: html2ans.parsers.base.ParseResult

A wrapper for parsing results. The idea of a parsing “match” is necessary so that we can try multiple parsers per tag (and not try multiple parsers when we don’t have to). For example, when parsing <p></p>, if we only returned an empty dictionary with the first available parser, the next logical step is to try the next parser. By returning match=True in that situation, we don’t make any more parsing attempts, we just move on to the next element in the tree.

Parameters:
  • output (dict or list[dict]) – The output ANS
  • match (bool) – Whether or not this parser was a match for a given element
